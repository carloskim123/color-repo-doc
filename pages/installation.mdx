# Installation

Color Repo Kim is a lightweight JavaScript library designed for seamless color manipulation and analysis. To begin using Color Repo Kim in your project, follow the simple installation steps below.

## Prerequisites

Before installing Color Repo Kim, ensure that you have Node.js and npm (Node Package Manager) installed on your machine. You can download and install them from the official [Node.js website](https://nodejs.org/).


Open your terminal and run the following command to add Color Repo Kim to your project:

```bash
npm install color-repo-kim
```

This command will download and install Color Repo Kim along with its dependencies.

Verify Installation
To verify that Color Repo Kim is successfully installed, create a simple JavaScript file and import the library. Here's a quick example:

```javascript
// sample.js
import { ColorRepoKim } from 'color-repo-kim';

console.log('Color Repo Kim Example:', ColorRepoKim.convert.hexToRgb("#000"));

```

If the installation is successful, you should see the result printed in the console.

Next Steps
Now that you have Color Repo Kim installed, you're ready to explore its color manipulation features. Check out the Color Repo Kim Documentation for in-depth guides and examples.
